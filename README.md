![image](https://github.com/justafolk/Consumer-Electronics-Ecommerce-system/assets/65607123/5aeb6a6d-2639-4b4e-8b33-2b1c1345d9af)# Consumer Electronics Ecommerce System

This is a web application made using PHP, HTML, CSS, Bootstrap, JS, and Python. The project allows authentication using Google Auth, has an active payment gateway, and allows administrators, sellers, and consumers to operate accordingly. The database is hosted on Azure MySQL Server, utilizing MySQL. The installation requires PHP 7.4 and Apache serverâ€”a LAMP stack with MySQL enabled.
